Проект: Система управления отелем
Описание
Этот проект представляет собой систему управления отелем, позволяющую бронировать номера, вести учёт гостей и анализировать популярность номеров.

Шаги по установке и запуску
Установка Python

Загрузите и установите Python (версия 3.10 или новее) с официального сайта: https://www.python.org.

Во время установки убедитесь, что вы отметили галочку "Add Python to PATH".

Создание виртуального окружения

Откройте командную строку (Command Prompt) и перейдите в папку проекта:

cmd
cd путь_к_проекту
Создайте виртуальное окружение:

cmd
python -m venv venv
Активируйте окружение:

Для Windows:

cmd
venv\Scripts\activate
Установка зависимостей

Убедитесь, что файл requirements.txt находится в папке проекта.

Установите все необходимые зависимости:

cmd
pip install -r requirements.txt
Инициализация базы данных

Выполните следующую команду для создания базы данных:

cmd
python init_db.py
Запуск проекта

Запустите приложение:

cmd
python app.py
Откройте браузер и перейдите по адресу:

http://127.0.0.1:5000
Использование
Бронирование номеров: Через вкладку "Bookings" можно добавлять, редактировать и отменять бронирования.

Управление гостями: Позволяет вести учёт клиентов.

Анализ статистики: Доступен отчёт о занятости номеров и популярности.

Требования
Python 3.10 или новее
